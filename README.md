# 燃豆积分俱乐部 SDK for JAVA

## Requirements

```
"JAVA": ">=1.8"
```


## Installment

### Maven
```
<dependency>
  <groupId>com.randou-tech</groupId>
  <artifactId>randou_java_sdk</artifactId>
  <version>1.4.0</version>
</dependency>
```

### Gradle

```
implementation 'com.randou-tech:randou_java_sdk:1.4.0'
```
